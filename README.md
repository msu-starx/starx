## Installation

First...

```
sudo apt-get update && sudo apt-get upgrade
```

Second...

```
sudo apt-get install build-essential
```

Third...

```
cd ~
git clone https://github.com/msu-starx/starx.git
```

Lastly...

```
cd starx
make
```

Run...

```
./starx
```
